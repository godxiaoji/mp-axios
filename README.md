# mp-axios

基于axios加入小程序适配器，同时实现cookie的扩展

## Install

使用NPM的方式引入：

```bash
npm install mp-axios
```

微信小程序需要单独构建：[npm 支持](https://developers.weixin.qq.com/miniprogram/dev/devtools/npm.html)

`const mpAxios = require('mp-axios');`

也可以使用`dist`目录下的构建版本

### Note

暂时不支持upload
